1.  Home will go to the main page, index.php.

 

2. Libraries will go to the LibraryList.php page.  

2.A. All users will see a Books button when a library is clicked on.

2.B. An admin logged in will see the above additional buttons when a library is clicked on.

 

 

2.B.a. Update Images button will go to the Edit Library Images page, LibraryImageUpdater.php. From here an admin can edit or delete a library image.

 

       OR

2.B.b. The Books button will go to the libraryInventory.php page, where all the books will be displayed from that library. Anyone can search for a title, author, or publisher. Only an admin will have the Upload Images button where they can edit the book images.

2.B.c. The Export Library button will export all the books from that library to an excel spread sheet.

2.B.d. The Import Books button will allow the libraries to be imported from an excel sheet, similar to the exported excel sheet. NOTE: by default, the Delete existing inventory will be selected. BE CAREFUL this will delete all the inventory from the library!

 

2.B.e. The Create a Book button will bring you to the BookEdit.php screen in order to add a new book to a library.

 

2.B.f. The Edit Library Details button will bring you to the LibraryEdit.php screen in order to edit the current library details.

 

3. The Blog link will go to the blog.php site. Only Admin will have the above 2 buttons.

4.A. Sponsors link will list all sponsors either by list view or grid view. You can click on each sponsor to bring you to the sponsors.php page, which will show what projects each sponsor is working on.

4.B. Projects link will list all projects either by list view or grid view. You can click on each project to bring you to the projects.php page, which will show what sponsors are working on this project.

 

 

5. About link will go to the about.php page. This has some information about the page and the admin can update the images.

 

 

5. Contact link will go to the Contact information page that will allow you to send an email to.

 

6. Reports link will go to the reports page and allow the Admin to export the data to an excel spread sheet.

 

 

7. Login link will go to a login screen. There are normal user logins and admin logins. The Login link will turn to a logout link if successful.

 

8. Admin link will show up if logged in as admin. It will bring you to the above site.

 

8.A. Libraries button will go to the library_list.php page. This will list the libraries as seen below.

 

 

 

 

8.B. Books button will go to the book_list.php page. This will list the books as seen below.

 

8.C. Projects button will go to the project_list.php page. This will list the projects as seen below. From here an admin can Add, Edit, or Delete a project.

 

8.D. Sponsors button will go to the sponsor_list.php page. This will list the sponsors as seen below. From here an admin can Add, Edit, or Delete a sponsor.

 

8.E. Sponsorships button will go to the sponsorships_list.php page. This will list the sponsorships as seen below. From here an admin can Add, Edit, or Delete a sponsor. Here a sponsor will be assigned to a project.

8.F. Members button will go to the UserManagement.php page. This will allow the admin to add, delete, or logout users.

 

8.G. Configure button will go to the Configure.php page. This will allow the admin to edit all the website settings and configurations.

8.H. Backup button will create a backup<todaysdate>.sql file, downloading all the tables from the website.

 

8.I. Exceptions button will go to the exceptions_list.php page as seen below. This will display all books that are missing pages, book image, Year Published, or Author.

 

8.J. Export button will export all the books from the website to a excel spreadsheet.

 

8.K. Import button will go to the importer.php page as seen above in step 2.B.d. Remember that delete existing inventory is set by default.

 

8.L. Help button will bring you to this current screen.
